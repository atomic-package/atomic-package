## Breadcrumb Components

Atomic-Packageで用意されている、ベースとなるパンくずリストStyleの一覧です。

<h3 class="title article sub">Example</h3>

<nav class="breadcrumb">
  <ul class="breadcrumbList">
    <li>
      <a href="">Home</a>
    </li>
    <li>
      <a href="">Blog</a>
    </li>
    <li class="disable">
      <a href="#">Category</a>
    </li>
    <li class="active">
      <a href="#">Post</a>
    </li>
  </ul>
</nav>

<h3 class="title article sub">Markup</h3>

```html
<nav class="breadcrumb">
  <ul class="breadcrumbList">
    <li>
      <a href="">Home</a>
```
