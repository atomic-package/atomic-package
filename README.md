## Quick start

**Clone the repo:**
```
git clone git@github.com:atomic-package/atomic-package.git
```

**Install with [Bower](http://bower.io):**
```
bower install atomic-package
```

**Install with [npm](https://www.npmjs.com):**
```
npm install @atomic-package/core
```
