# Atomic Package
## APB CSS (Atomic Parts Base CSS) + JavaScript framework.

**APB CSS**
http://apbcss.com/

## Quick start

**Clone the repo**
`git clone git@github.com:webcyou/atomic-package.git`

**Install with Bower**
'bower install atomic-package'

*Install with npm*
'npm install atomic-package'
